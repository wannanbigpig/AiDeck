# AiDeck 更新记录

本文档用于永久记录每次版本更新内容。消息中心的 `announcements.json` 只保留当前需要提示的最新版本更新公告；发布下一个版本时，应删除或替换上一版本的更新公告，但不要删除本文档中的历史记录。

## v1.0.4

- Codex 设置新增 CLI 命令位置配置，支持自动检测、手动选择和自定义输入，后续账号启动、继续会话和唤醒任务统一使用该命令。
- Codex CLI 设置中显示检测到的完整版本输出，便于确认当前使用的 `codex-cli` 版本。
- 修复从 AiDeck 启动 Codex CLI 时，绝对 `codex` 路径仍可能被 `PATH` 中旧 Node 执行的问题，并精简终端启动命令中的环境变量。
- 立即唤醒改为后台异步执行，点击后立即显示“唤醒中”状态，并记录最近结果、耗时、上次执行、下次触发和触发方式。
- Codex 唤醒使用独立 `CODEX_HOME` 和独立会话来源，避免唤醒任务读取普通账号实例规则或污染普通会话。
- 优化 Codex 会话管理：默认实例历史会话与 Codex 桌面项目列表口径对齐，唤醒会话统一聚合展示，顶部工具栏控件高度对齐。
- 同步更新项目版本号、README 版本徽章和公告内容。

## v1.0.3

- 新增后台任务运行提醒，说明定时唤醒、自动刷新等任务依赖 uTools 插件保持后台运行。
- 后台任务提醒改为放入消息通知中心，不再占用首页仪表盘空间。
- 消息通知中心的全部已读、刷新、关闭操作移到标题栏右侧，并统一为紧凑图标按钮。
- 刷新按钮增加加载旋转反馈，点击后至少展示 1 秒加载状态。
- Codex CLI 账号绑定实例不再共享会话数据，仅共享 `skills`、`rules`、`vendor_imports/skills` 和 `AGENTS.md`，避免多实例会话索引与 `state_5.sqlite` 状态不一致。
- 新增 Codex 会话管理入口，可在账号页切换到会话视图，按工作区查看默认实例和账号绑定实例中的本地会话。
- 支持预览 Codex 会话消息，Markdown 内容、代码块、表格和图片附件可直接查看，内部指令和工具日志默认折叠到调试信息中。
- 支持继续会话、归档 / 取消归档、移入回收站、从回收站恢复或彻底删除会话。
- 支持将会话复制或移动到指定账号绑定实例，并在复制时生成新的会话 ID，避免多账号实例复用同一会话索引。
- 新增会话索引清理能力，可移除缺失会话文件残留的 SQLite / `session_index.jsonl` 索引记录。
- 使用说明从页面内折叠卡片改为标题旁弹窗入口，侧边栏默认收起，设置页返回会回到进入设置前的平台。
- 同步更新项目版本号、README 版本徽章和公告内容。

## v1.0.2

- 项目收敛为 uTools 单端，移除 Electron Desktop、旧根入口和历史 preload 转发层。
- 新增消息通知中心，支持远程公告、未读角标、置顶、刷新、全部已读和自动弹窗。
- Codex / Gemini 账号卡片新增 CLI 启动按钮，并支持默认终端设置。
- Codex CLI 启动改为账号绑定独立实例，首次点击创建独立 `CODEX_HOME`，后续复用；删除账号时同步删除绑定实例。
- Codex CLI 多账号实例共享静态配置目录，便于复用规则和 skills。
- Codex 账号卡片新增唤醒任务按钮，支持按账号设置启用 / 停用、每日 / 每周 / 间隔 / 配额重置 / 启动后调度、预设或自定义模型，并可立即用账号绑定实例执行 `codex exec` 轻量唤醒；模型预设支持 `GPT-5.5`、`GPT-5.4`、`GPT-5.4-Mini`、`GPT-5.3-Codex`、`GPT-5.2`，Pro 账号额外显示 `GPT-5.3-Codex-Spark`。
- Codex 配额展示补齐 5 小时、每周、剩余额度和额外模型额度。
- Codex 卡片背面新增订阅到期展示。
- 配额自动刷新间隔支持 1-60 分钟细粒度设置。
- 新增系统级配额预警通知。
- 优化请求日志写入，降低高频日志场景下的同步 IO 压力。
- 修复操作日志弹窗懒加载边界、批量刷新返回值、导入按钮文案等问题。

- 修复生产环境下应用内 logo 资源路径异常。

## v1.0.0

- 建立 uTools 插件形态的多平台账号管理主流程。
- 支持 Antigravity、Codex、Gemini CLI 账号导入、切换、配额查看和标签管理。
- 引入本地数据目录、日志轮转、OAuth 回调和基础同步能力。
